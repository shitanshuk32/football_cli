# Football CLI

## Vision

A developer-first football application that runs entirely from the terminal.

Users can get:

- Live scores
- Today's matches
- Team information
- AI summaries (future)
- Match analysis (future)

without opening a browser.

## Why Am I Building This?

To learn:

- Backend Engineering
- APIs
- PostgreSQL
- AI Integrations
- CLI Development
- Deployment

through one real project.
